СКИДКИ

Поиск скидки по ID
GET http://localhost:8080/api/v1/bonus_system/discount/1/id
Список всех скидок
GET http://localhost:8080/api/v1/bonus_system/discount/list
Добавить новую скидку
POST http://localhost:8080/api/v1/bonus_system/discount Content-Type: application/json { "title": "SILVER", "startBonusSum": 50000.00, "rate": 3, "isActive": true }
Изменить скидку по ID
PUT http://localhost:8080/api/v1/bonus_system/discount Content-Type: application/json { "id": 1, "title": "GOLD", "startBonusSum": 50000.00, "rate": 3, "isActive": true }

ПАЦИЕНТЫ

Поиск пациента по ID
GET http://localhost:8080/api/v1/bonus_system/patient/1/id
Список всех пациентов
GET http://localhost:8080/api/v1/bonus_system/patient/list
Добавить нового пациента
POST http://localhost:8080/api/v1/bonus_system/patient Content-Type: application/json { "name": "Viserys", "bizBoxPatientId": 228, "accumulatedAmount": 50000, "isActive": true, "discounts": { "id": 2 }, "rate": 3 }
Изменить пациента по ID
PUT http://localhost:8080/api/v1/bonus_system/patient Content-Type: application/json { "id": 2, "name": "Vi", "bizBoxPatientId": 228, "accumulatedAmount": 50000, "isActive": true, "discounts": { "id": 1 }, "rate": 3 }
